Introduction

Thank you for purchasing our Startup - Startup Company Website Template. In this manual we will show you how to manage this template, edit it, set up and create a ready-to-use website. We did our best to simplify this instruction and make it clear and easy to use.


Software

Before proceeding with this Website Template installation and customization, you need to prepare. We recommend that you download all the required software to edit this Website Template. Software Required is listed below:

1. First, you need proper unzipping software for ZIP package extracting. You may use WinZip (Windows) and Stuffit Expander (Mac).

2. You need also code editor, i. e. Visual Studio Code, Sublime Text, Notepad++, Atom, etc. to edit the source code of the template files.

3. You might also require Adobe Photoshop. It is required for editing template graphic design and images.

4. For files uploading to your hosting server FTP manager will be also needed, like Total Commander, FileZilla, CuteFTP, etc.

HTML & CSS File Editing

All of the HTML files are located at "startup" folder (index-1.html, index-2.html, etc.). Every HTML file represents separate template page.

Visual Studio Code is an advised application for HTML files content editing, but you may also use any other suitable software.

In order to open HTML file with Visual Studio Code software, click on the file with right mouse button and select Open with > Visual Studio Code in the dropdown menu. After file opening you can edit it.

CSS files are used to change template appearance, i.e., fonts (font-family, size, font-face, etc.), color, background, width, etc. CSS files can be edited with Visual Studio Code software.

Text Editing

As previously stated, all site content is inserted to HTML files. You'll need minimal skills on orienting within code.

In order to replace “lipsum” (filling text) with your own one, please do the following:

1. Navigate to "startup" folder within template package.

2. Open index-1.html file with HTML editor.

3. You may use search function (CTRL+F) to check for required text.

4. Edit text with HTML editor. Save changes and open your file in browser to see them applied.

Image Editing

Navigate to 'startup/img' folder. First way is to upload new images keeping the original filenames and extensions, simply replacing them.

Second way is to upload custom images specifying your own filenames and extensions.

Next you'll also need to change image filename in HTML file. Please do the following:

1. Open HTML file within startup folder by using HTML editor for this purpose.

2. Use search function (CTRL+F) to locate an image in HTML file.

3. Edit image filename and extension. Save changes and open your file in browser to see them applied.

Contact Form Activation

Startup comes with a fully working and functional Contact Form with Ajax & PHP.

To make the Contact Form working, simply go to "mail" folder and open "contact-form.php" file with a text editor. Look out for "info@example.com", replace your email with this and save the file.

Note: Our Contact Form is compatible with PHP 5.5 and later, including PHP 8.0

Uploading

You can run HTML file directly from Startup folder, but it won't let you explore all template functionality. You should upload all the content of "startup" folder from your local computer to your hosting server in order to display a fully working website and make your site "live".

For files uploading to your hosting server FTP manager will be needed, like Total Commander, FileZilla, CuteFTP, etc.

In case you have existing server already, please check if it meets all the requirements. For most templates PHP 5.4+ version is the only one required.

Your website root depends on the directory structure on your hosting server. If you upload the "startup" folder itself into your server, the root to your website will be "http://your_domain_name/startup". To avoid this and make the website root "http://your_domain_name/" please open the "startup" folder and upload the folder content.
